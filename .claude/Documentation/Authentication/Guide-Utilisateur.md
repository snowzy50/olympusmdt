# Guide Utilisateur - Connexion Discord

## Pour les Utilisateurs

### Comment se connecter ?

1. **Accédez à la page de connexion**
   - Rendez-vous sur : `https://votre-mdt.com/login`

2. **Cliquez sur "Se connecter avec Discord"**
   - Vous serez redirigé vers Discord
   - Discord vous demandera d'autoriser l'application

3. **Autorisez l'application**
   - Cliquez sur "Autoriser"
   - Vous serez automatiquement redirigé vers OlympusMDT

4. **Sélectionnez votre agence**
   - Vous verrez uniquement les agences auxquelles vous avez accès
   - Cliquez sur l'agence souhaitée

5. **Accédez au Dashboard**
   - Vous êtes maintenant connecté à votre MDT !

## Rôles Discord Requis

Pour accéder à OlympusMDT, vous devez avoir **au moins un** des rôles suivants sur Discord :

| Rôle Discord | Agence | Description |
|--------------|--------|-------------|
| MDT SASP | San Andreas State Police | Police d'État |
| MDT LSPD | Los Santos Police Department | Police de Los Santos |
| MDT BCSO | Blaine County Sheriff Office | Shérif du Comté |
| MDT EMS | Emergency Medical Services | Services Médicaux |
| MDT DOJ | Department of Justice | Département de la Justice |

## Questions Fréquentes

### Je n'arrive pas à me connecter

**Vérifiez que :**
- Vous êtes bien membre du serveur Discord Olympus RP
- Vous avez au moins un rôle MDT sur Discord
- Votre compte Discord est vérifié

### Je ne vois pas toutes les agences

**C'est normal !** Vous ne voyez que les agences correspondant à vos rôles Discord.

**Exemple :** Si vous avez le rôle "MDT SASP" sur Discord, vous ne verrez que l'agence SASP.

### Comment obtenir l'accès à une agence ?

Contactez un administrateur sur Discord pour obtenir le rôle MDT correspondant.

### Je vois "Vous n'avez pas les rôles requis"

Cela signifie que vous n'avez aucun rôle MDT sur Discord. Contactez un administrateur.

### Comment me déconnecter ?

1. Sur la page de sélection d'agence, cliquez sur "Se déconnecter"
2. Vous serez redirigé vers la page de connexion
